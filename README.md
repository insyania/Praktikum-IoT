# Praktikum-IoT — Internet of Things with Arduino

> 📚 Practicum from **Internet Engineering Technology**, Universitas Gadjah Mada  
> 🛠️ Tools: Arduino Kit, Arduino IDE


## 📖 About This Repository

This repository documents hands-on IoT practicum sessions at UGM, where we built simple embedded systems and IoT projects using Arduino hardware and software. Each session followed guided modules provided by the lab instructor, focusing on real-world sensor-actuator integration.


## 🧠 Topics & Skills Covered

**Hardware**
- Arduino Uno / Nano microcontroller
- Digital and analog I/O pins
- Sensors: temperature, light (LDR), ultrasonic (HC-SR04), humidity (DHT11)
- Actuators: LEDs, buzzers, servo motors, relays

**Programming**
- Arduino C/C++ syntax
- `setup()` and `loop()` structure
- Serial Monitor for debugging
- Library management in Arduino IDE

**IoT Concepts**
- Sensor data acquisition and processing
- Hardware-software integration
- Embedded systems logic and control flow


## 🛠️ Tools & Environment

| Tool | Usage |

| Tinkercad (simulation) | Circuit simulation (optional) |


## 🚀 How to Run

1. Install [Arduino IDE](https://www.arduino.cc/en/software)
2. Connect Arduino board via USB
3. Open any `.ino` file from this repo
4. Select the correct board and port: `Tools → Board` and `Tools → Port`
5. Click **Upload** (→)


## 💡 Key Learning Outcomes

- Understands the hardware-software interface in embedded systems
- Able to write and deploy code to microcontrollers
- Builds foundational IoT knowledge relevant to smart systems and network-connected device security
